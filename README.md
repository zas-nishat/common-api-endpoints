# Common API Endpoints Collection 🚀

A comprehensive collection of **11,968+** unique API endpoints, curated and sorted for developers, security researchers, and testers. This list is designed to assist in API discovery, fuzzing, and testing environment setups.

## 📌 Features
* **Total Endpoints:** 11,968+
* **Format:** Standard `.txt` file with one endpoint per line.
* **Pre-formatted:** All endpoints are prefixed with `/` for direct integration.
* **Sorted:** Organized in **Ascending (A-Z)** order for easy navigation.
* **Cleaned:** No duplicates, empty lines, or trailing whitespaces.
